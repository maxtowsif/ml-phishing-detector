> **AI-Powered Phishing Detection System**

A real-time web application that uses machine learning to detect phishing websites through advanced URL analysis and Random Forest classification.

![Cybersecurity](https://www.rmmagazine.com/images/default-source/magazineimages/2021/07-08/rm7-8-21_find_cybersecurity.jpg?sfvrsn=59dc887f_6)

## ▶ Features

- ▸ **Real-time Phishing Detection** - Instant URL analysis with confidence scoring
- ▸ **Machine Learning Powered** - Random Forest classifier with 30+ features
- ▸ **User Authentication** - Secure login with user/admin roles
- ▸ **Interactive Dashboard** - Visual confidence scores and analytics
- ▸ **Feedback System** - User feedback collection and analysis
- ▸ **Responsive Design** - Mobile-friendly web interface

## ◉ Tech Stack

- **Frontend**: Streamlit
- **Backend**: Python 3.9+
- **ML**: Scikit-learn, Random Forest
- **Data**: Pandas, NumPy
- **Visualization**: Matplotlib, Seaborn

## ► Quick Start

### Prerequisites
- Python 3.9+
- pip

### Installation & Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/maxtowsif/ml-phishing-detector.git
   cd ml-phishing-detector
   ```

2. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

3. **Run the application**
   ```bash
   streamlit run app.py
   ```

4. **Open in browser**
   - Local: `http://localhost:8501`
   - Network: `http://YOUR_IP:8501`

## ◆ How to Use

1. **Login** - Enter your credentials on the welcome page
2. **Enter URL** - Paste any website URL you want to check
3. **Get Results** - View phishing detection results with confidence scores
4. **Provide Feedback** - Rate the accuracy of predictions to help improve the system

## ◈ Machine Learning

- **Model**: Random Forest Classifier
- **Features**: 30+ URL-based security indicators
- **Training**: Comprehensive phishing dataset
- **Accuracy**: High-performance classification with confidence scoring
